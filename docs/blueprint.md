# **App Name**: MiniHeroes Habit Builder

## Core Features:

- Admin Authentication: User authentication for Admin Masters (parents/guardians) via email/password and Google using Firebase Authentication.
- Child Access: Child profile creation and login using a unique 6-digit access code. Login uses Firebase Custom Tokens for secure access.
- Family Management: Family collaboration feature allowing multiple adults to manage a child's profile through invite codes and role-based access.
- Gamified Task Tracking: Gamified task management with rewards, dreams, and progress tracking (stars, XP, levels) to motivate children.
- AI Task Suggestions: AI-powered task suggestion tool to help parents brainstorm appropriate tasks based on the child's age, interests, and current progress.
- Habit Management Interface: UI components for managing tasks, rewards, and dreams with a focus on accessibility and child-friendly design.

## Style Guidelines:

- Primary color: Cheerful light blue (#64B5F6), evoking a sense of calmness and encouragement.
- Background color: Very light blue (#E3F2FD), for a clean and calming interface.
- Accent color: A contrasting yellow-orange (#FFB74D) to highlight important actions and rewards.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern, friendly feel.
- Use clear, playful icons that represent tasks, rewards, and dreams. Icons should be easily recognizable by children.
- Clean and intuitive layout with a focus on ease of navigation for both parents and children. Implement clear visual hierarchy and consistent spacing.
- Subtle animations and transitions to provide visual feedback and enhance the user experience, especially when completing tasks or earning rewards.
